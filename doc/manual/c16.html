Capítulo 2. Secciones

En esta sección se describirán brevemente los distintos módulos que componen la página principal de la reserva de vuelo. Servirán como referencia para el Capítulo 3.

2.1. Usuario

En este apartado el usuario puede loggearse si ya es un usuario registrado. Para hacerlo deberá ingresar los siguientes datos provistos en la fase de registro:

  • Email

  • Contraseña

Después de ralizar este proceso, o si el usuario ya se había registrado en una página anterior, esta sección se transformará como muestra la Figura 2-2, con un mensaje de bienvenida. Además los datos de la Sección 2.3 se autocompletarán con los datos del usuario.
